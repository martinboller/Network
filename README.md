# Simple BASH script to capture packets remotely

## Usage:
.\WoS.sh RemoteHost RemoteInterface RemotePort RemoteUser RemoteKey1
## Example:
./WoS.sh server1 eth0 10022 root ~/.ssh/id_ed25519"  
If everything else is in place, try  
./WoS.sh server1 eth0
